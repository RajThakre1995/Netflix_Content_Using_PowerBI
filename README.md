# 🎬 Netflix Content Analytics Dashboard | Power BI

## 📌 Project Overview
This project presents an interactive **Netflix Content Analytics Dashboard** built using Power BI.  
The goal of this project is to analyze Netflix’s content library and uncover insights related to content growth, genre popularity, and rating distribution.

The dashboard transforms raw Netflix data into meaningful visual insights to support data-driven decision-making.

---

## 🎯 Objective
- Analyze Netflix content distribution across Movies and TV Shows  
- Track content growth trends over the years  
- Identify top-performing genres  
- Understand rating distribution patterns  
- Build an interactive and user-friendly dashboard  

---

## 📊 Key Insights
- Total Titles analyzed: **6,234**
- Movies dominate the platform (~68%)
- Rapid content growth observed after **2015**
- Average movie duration is **~99 minutes**
- Documentaries and Stand-Up Comedy are top genres
- Majority of content falls under **TV-MA** and **TV-14**

## 📈 Dashboard Features
- ✅ KPI Cards (Total Titles, Movies, TV Shows, Avg Duration)  
- ✅ Content Growth Over Time (Line Chart)  
- ✅ Movies vs TV Shows Distribution (Donut Chart)  
- ✅ Top 10 Genres Analysis (Bar Chart)  
- ✅ Rating Distribution (Column Chart)  
- ✅ Interactive Slicers:
  - Type  
  - Release Year  
  - Country  
  - Rating

## 🏗️ Data Processing Steps
1. Imported raw Netflix dataset into Power BI  
2. Cleaned and transformed data using Power Query  
3. Created data model and relationships  
4. Built DAX measures for KPIs  
5. Designed interactive dashboard visuals  
6. Applied Netflix-style dark theme UI
